# NayePankh Foundation Webpage

A responsive and modern website developed for NayePankh Foundation, a youth-led NGO dedicated to empowering underprivileged communities through education, women empowerment, hygiene awareness, and hunger relief initiatives.

## Features

* Responsive design for mobile, tablet, and desktop devices
* Interactive navigation and smooth scrolling
* Dark Mode support
* Animated sections and hover effects
* About Us, Programs, Volunteer, and Contact sections
* Clean and user-friendly interface
* Modern HTML, CSS, and JavaScript implementation

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Purpose

This project aims to create an engaging online presence for NayePankh Foundation, showcasing its mission, core initiatives, and opportunities for volunteering while providing visitors with an intuitive and accessible browsing experience.
